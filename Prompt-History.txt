Generate a WebOracle Project which is a magic oracle ball that answers the users answers with fancy animations that at no point should stutter because it is optimized.
This project is build with a standard backend- and frontend architecture. The fontend is angular while the backend is a spring-boot api rest based backend. Install all needed tools such as angular cli etc. The Frontend should be visually very pleasing and very simple to handle, it should have a very easy central input field in the middle where the user can insert his question, after submitting the question - the oracle will proceed with a beautiful waiting animation of about random 3-12 seconds which is specifically only delayed on the frontend also it can be random for a better engagement feeling. The answer is displayed in a short form and a long form which can be accessed by clicking on the message. After that a longer answer is given and there will be a recommendation store with products that may apply to the person and may help in her situation. For that we need a mssql database with amazon product-links, as well as categories for certain questions/ problems. the product links are placeholders for later affiliate links. the links should be imbedded by amazon-affiliate image embedding.
